# 3D programming

- [Alice](http://www.alice.org/index.php?page=what_is_alice/what_is_alice) Alice is a 3D programming environment that makes it easy to create an animation for telling a story, playing an interactive game, or a video to share on the web. (take care that Alice 2 is not free and Alice 3 has the old three clause MIT license; parts of the art gallery provided are not free)
